# Markdown Features in Bengal

Bengal supports all standard Markdown features plus some extras.

## Headings

Use `#` for headings from H1 to H6.

## Code Blocks

```python
def hello():
    print("Hello, World!")
```

## Lists

- Unordered lists
- Use dashes or asterisks
- Nested lists work too
