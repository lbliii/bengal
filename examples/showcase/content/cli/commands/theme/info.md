# info

Show theme info for a slug (source, version, paths).


## Usage

```bash
bengal theme info [ARGUMENTS]
```

## Arguments

### slug

**Type:** `text`
**Required:** Yes
**Default:** `Sentinel.UNSET`

### source

**Type:** `path`
**Required:** No
**Default:** `.`





## Help

```bash
bengal theme info --help
```
