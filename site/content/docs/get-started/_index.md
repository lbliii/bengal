## Features

- Parallel and incremental builds with optional free-threaded Python support
- Native Python tooling, no Node.js required (unless you want it)
- Hot reload, CLI wizards, and auto-generated API docs
- Support for blogs, docs, portfolios, resumes, and custom setups via interactive wizard
