Version history and release notes for Bengal.

## Versioning

Bengal follows [Semantic Versioning](https://semver.org/):

- **Major** (1.x.x): Breaking changes to public APIs
- **Minor** (x.1.x): New features, backward compatible
- **Patch** (x.x.1): Bug fixes, backward compatible

## Current Version

The current stable release is **0.1.7** (January 2026).

## Release Schedule

Bengal releases when features are ready. There is no fixed release schedule during the 0.x development phase.
