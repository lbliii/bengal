# perf
**Type:** Command
**Source:** [View source](bengal/bengal/cli/commands/perf.py#L11)

```{badge} Command
:class: badge-secondary
```



**Command:** `bengal.utils.perf`


Show performance metrics and trends.

Displays build performance metrics collected from previous builds.
Metrics are automatically saved to .bengal-metrics/ directory.

## Usage

```bash
perf [OPTIONS] [ARGUMENTS]
```

## Options

### `last`


Show last N builds (default: 10)

**Type:** `integer`

**Default:** `10`




### `format`


Output format

**Type:** `choice`

**Default:** `table`




### `compare`


Compare last two builds

**Type:** `boolean`

**Default:** `False`
