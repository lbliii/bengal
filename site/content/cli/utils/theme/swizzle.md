# swizzle

Copy a theme template/partial to project templates/ and track provenance.


## Usage

```bash
bengal utils theme swizzle [ARGUMENTS]
```

## Arguments

### template_path

**Type:** `text`
**Required:** Yes
**Default:** `Sentinel.UNSET`

### source

**Type:** `path`
**Required:** No
**Default:** `.`





## Help

```bash
bengal utils theme swizzle --help
```
