# info

Show theme info for a slug (source, version, paths).


## Usage

```bash
bengal utils theme info [ARGUMENTS]
```

## Arguments

### slug

**Type:** `text`
**Required:** Yes

### source

**Type:** `path`
**Required:** No
**Default:** `.`





## Help

```bash
bengal utils theme info --help
```
