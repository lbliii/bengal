## Options

Content sections to create (e.g., blog posts about). Default: blog

**Type:** `text`

### `with_content`

**Type:** `boolean`

**Default:** `False`





### `pages_per_section`


Number of sample pages per section (with --with-content)

**Type:** `integer`

**Default:** `3`





### `dry_run`


Preview what would be created without creating files

**Type:** `boolean`

**Default:** `False`





### `force`

**Type:** `boolean`

**Default:** `False`
