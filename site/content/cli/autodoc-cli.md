**Command:** `bengal.autodoc-cli`


⌨️  Generate CLI documentation from Click/argparse/typer apps.

Extracts documentation from command-line interfaces to create
comprehensive command reference documentation. Useful for documenting
custom CLI tools or generating standalone CLI docs.

## Usage

```bash
autodoc-cli [OPTIONS] [ARGUMENTS]
```

## Options

### `app`


CLI app module (e.g., bengal.cli:main)

**Type:** `text`





### `framework`


CLI framework (default: click)

**Type:** `choice`

**Default:** `click`




### `output`


Output directory for generated docs (default: content/cli)

**Type:** `path`





### `include_hidden`


Include hidden commands

**Type:** `boolean`

**Default:** `False`




### `clean`


Clean output directory before generating

**Type:** `boolean`

**Default:** `False`




### `verbose`


Show detailed progress

**Type:** `boolean`

**Default:** `False`




### `config`


Path to config file (default: bengal.toml)

**Type:** `path`
