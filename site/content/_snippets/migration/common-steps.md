1. **Install Bengal** — `pip install bengal` or `uv add bengal`
2. **Create a new site** — `bengal new site mysite`
3. **Copy content** — transfer markdown files to `content/`
4. **Convert syntax** — replace platform-specific syntax with Bengal directives
5. **Update configuration** — convert config files to `bengal.toml` and `config/`
6. **Test and verify** — run `bengal build` and `bengal check`
