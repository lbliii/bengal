The default theme's color system is now authored in OKLCH for wide-gamut richness, and the five built-in palettes are defined as a handful of anchors whose hover, active, border, surface, and state variants derive automatically via relative-color and ``color-mix`` — retuning a palette no longer means editing a hand-painted hex matrix. The global focus ring is now palette-aware (it previously fell back to a fixed Material blue in every non-default palette).
