**Command:** `bengal.fix`


ðŸ”§ Auto-fix common health check issues.

Analyzes health check results and applies safe fixes automatically.
By default, only applies SAFE fixes (reversible, no side effects).

## Usage

## Options

### `validator`


Only fix issues from specific validator (e.g., 'Directives', 'Links')

**Type:** `text`





### `dry_run`


Show what would be fixed without actually applying fixes

**Type:** `boolean`

**Default:** `False`




### `confirm`


Ask for confirmation before applying fixes

**Type:** `boolean`

**Default:** `False`




### `fix_all`


Apply all fixes including those requiring confirmation

**Type:** `boolean`

**Default:** `False`
