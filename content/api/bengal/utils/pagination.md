## Classes




### `Paginator`


Paginator for splitting a list of items into pages.

Usage:
    paginator = Paginator(posts, per_page=10)
    page = paginator.page(1)  # Get first page



**Attributes:**

:::{div} api-attributes
`items`
: List of items to paginate

`per_page`
: Number of items per page

`num_pages`
: Total number of pages
