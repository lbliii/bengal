


Context manager for simple progress feedback in CLI commands.


**Parameters:**

| Name | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `description` | `str` | - | Description text for the progress task |
| `total` | `int \| None` | - | Total number of items (None for indeterminate) |
| `cli` | `CLIOutput \| None` | - | Optional CLIOutput instance (creates new if not provided) |
| `enabled` | `bool` | `True` | Whether to show progress (auto-disabled for quiet/non-TTY) |







**Returns**


`Iterator[Callable[[int | None, str | None], None]]`
:::{rubric} Examples
:class: rubric-examples
:::


```python
with cli_progress("Checking environments...", total=len(environments)) as update:
        for env in environments:
            check_environment(env)
            update(advance=1, item=env)
```
